# Amethyst Gaming Optimizer v7.1

A comprehensive Windows batch utility for optimizing gaming performance.

## Features
- 18 optimization categories
- Service management
- Registry tweaks
- Network optimization
- GPU/CPU specific tweaks
- System debloating
- One-click complete optimization

## Usage
1. Right-click `AmethystOptimizer.bat`
2. Select "Run as administrator"
3. Choose options from the menu

## Warning
- Always create a restore point first (Option P)
- Some optimizations are aggressive
- Use at your own risk
